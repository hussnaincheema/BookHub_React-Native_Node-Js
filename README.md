## 📚 BookHub – Full Stack Book Management App

A modern mobile application built with React Native, Tailwind CSS (NativeWind), Node.js, and MongoDB.
This project demonstrates a complete CRUD (Create, Read, Update, Delete) system where users can manage their book collection.

The application includes secure user authentication and a REST API built with Express.js and MongoDB, allowing users to register, login, and perform book management operations such as adding, updating, viewing, and deleting books.

## 📱 App Sections

## 🔐 Authentication

The authentication section allows users to create accounts and securely log into the application.

## Features include:

User registration

Secure login with email and password

JWT-based authentication

Protected routes for authenticated users

## 📚 Book List

This section displays all books stored in the database.

## Features:

Fetch books from MongoDB

Display books using React Native FlatList

Dynamic data rendering

Real-time updates after CRUD operations

## ➕ Add Book

Users can add new books to the system by entering details such as:

Book title

Author name

Description

Category

## ✏️ Update Book

Users can edit existing book information when required.

Fields that can be updated include:

Title

Author

Description

Category

## 🗑 Delete Book

Users can remove books from the system with a delete option, allowing them to manage their book collection efficiently.

## 📞 API Integration

The application communicates with a Node.js Express backend API responsible for handling authentication and book management operations.

## Features:

RESTful API architecture

Secure JWT authentication

MongoDB database integration

Full CRUD functionality
